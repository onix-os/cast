# Acknowledgments

This repository builds on [AerynOS OS Tools](https://github.com/AerynOS/os-tools),
which began in 2023 as the Serpent OS package and system-management tooling
project. Ikey Doherty started the original repository and Moss implementation.
Cory Forsstrom made foundational contributions to the `.stone` format,
`stone_recipe`, Boulder, and Moss.

The original project later continued under AerynOS and has been shaped by
Jonas Platte, Joey Riches, Rune Morling, Reilly Brogan, Fabio Forni, and many
other contributors. Thank you to everyone who has contributed code, reviews,
testing, documentation, packaging policy, and design work.

The complete contributor record is preserved in the
[Git history](https://github.com/AerynOS/os-tools/commits/main) and
[contributors list](https://github.com/AerynOS/os-tools/graphs/contributors).
