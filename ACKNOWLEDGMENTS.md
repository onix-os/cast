# Acknowledgments

This hard fork exists because the Serpent OS and AerynOS communities built OS
Tools first and shared that work. Before Onix chose a different configuration
model, years of engineering had already gone into Moss, Boulder, the `.stone`
format, package transactions, repository handling, and the practical details
of building a Linux distribution. We inherited that work. We are sincerely
grateful for it.

Our first thanks go to **Ikey Doherty**, who started the original repository,
created Moss, and established much of the early package and system-management
architecture. Thank you for beginning the project, carrying it through its
formative years, and making it available for others to study and extend.

We are equally grateful to **Cory Forsstrom**. Cory's work runs through the
`.stone` format, `stone_recipe`, Boulder, Moss, and the integrity of package and
repository operations. A large part of what this fork can do rests on that
careful engineering.

Thank you to **Jonas Platte, Joey Riches, Rune Morling, Reilly Brogan, Fabio
Forni, Brendan Barnes, Philipp Trulson, Jonathan Lopez, Gavin Zhao**, and every
other contributor recorded in the history. They fixed bugs, reviewed designs,
improved diagnostics and portability, maintained repositories, tested real
transactions, wrote documentation, and did the quiet maintenance that keeps a
large toolchain usable.

We also thank the package maintainers and users who exercised these tools on
real systems. Bug reports, failed builds, design discussions, release testing,
and patient explanations to new contributors all became part of the project,
even when they did not appear as commits.

Hard-forking changes the direction of this repository; it does not erase its
origin. Onix is taking responsibility for an incompatible Gluon-only
configuration model. We are not claiming to have replaced the original work,
and we do not want the fork to hide who made it possible. We will preserve the
original authorship, license notices, and Git history, and we will keep
crediting upstream wherever this project is described.

The complete record belongs to the
[Git history](https://github.com/AerynOS/os-tools/commits/main) and
[contributors list](https://github.com/AerynOS/os-tools/graphs/contributors).
If we have missed someone or described a contribution poorly, please tell us.
That is an error to correct.
